# HC-SR04 Distance Logger  

📌 **Function**: Distance reading using ultrasonic method in cm and inch unit.  

## 🛠️ Hardware  
- Sensor: HC-SR04  
- Board: ESP32 (DOIT DEVKIT V1)
- Additional: Jumper cable/PCB board

## 📝 Wiring  
| Sensor |  Board  |  
|--------|---------|  
| VCC    | VIN     |  
| TRIG   | GPIO 13 |
| ECHO   | GPIO 14 |   
| GND    | GND     |
